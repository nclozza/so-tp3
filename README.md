# TP2 - Sistemas Operativos - 2018

## To run the project:

```
> git clone https://github.com/nclozza/so-tp2.git
> cd so-tp2/Toolchain
> make
> cd ..
> make
> ./run.sh
```

## For more instructions when the OS is running run the command `help`
